👌 第三步：获取模型、放大器、LoRA

   BrightProtoNuke（BPN）大模型是我最喜欢的 SDXL模型。

   它能够同时处理写实和动漫风格，并且可以在刚开始使用时获得非常好的结果。与大多数其他模型相比，它对您的提示也更具响应性。

   下载最新版本并将其放入文件夹中👇。

   ComfyUI/models/checkpoints/

   👀我建议使用以下放大器(UpScale)(可选)：

   4x NMKD Siax 用于摄影照片图像

   4x NMKD YandereNeo 用于动漫图像

   下载放大器(UpScale)并将它们放在文件夹中👇：

   ComfyUI/models/upscale_models/

   👀我建议使用以下 LoRA（可选）：

   Betterment XL：全面提升质量并去除不良阴影

   Cinematic Painting XL：为生成的图像添加电影般的大气效果。同时添加噪音。

   下载这些 LoRA 并将它们放入文件夹

   ComfyUI/models/LoRA/。

点击侧边栏上的“刷新”按钮。 现在，您将能够在“加载模型”节点中设置您的模型👇：

   在“CR Upscale Image（图像放大器）”节点中选择您的图像放大器模型👇：

同样在LoRA节点中选择你需要的LoRA模型👇：

👆这个节点，允许您设置多个 LoRA。switch_x 用于启用或禁用 LoRA，lora_name_x 用于选择 LoRA，而 model_weight_x 和 clip_weight_x 分别设置 LoRA 在主模型和 CLIP 模型上的强度。

   您的大模型包含一个主模型和一个 CLIP 模型，它们将分别对 LoRA 作出不同的响应，以下是这些差异的示例（主模型 = Unet 和 CLIP 模式 = 文本编码器）。

   🤔如果您感到困惑，不用担心。当您刚开始使用时，只需保持这些设置不变。
