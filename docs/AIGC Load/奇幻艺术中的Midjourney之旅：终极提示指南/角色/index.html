经典奇幻原型

   在创作奇幻故事时，有些人更喜欢采用以世界为先的方式：从构建世界、其传承和历史开始，然后考虑占据其中的角色。

   另一些人则更倾向于以角色为先的方式。从角色的个性和动机出发，逐渐构建整个世界。

   没有一种正确的方式。我认为你应该从吸引你的地方开始。话虽如此，我们将以一个奇幻角色原型的概述开始这本书。    这将让我们展示多种呈现角色的方法，以及用于表现它们的美学风格。

   它将涵盖角色的身体特征、服饰和盔甲。

   我们既会拥抱陈词滥调（在视觉和主题上），也会寻找摆脱它们的方法。

野蛮人

  野蛮人通常是一位原始而凶猛的战士，以其原始的力量和偶尔的狂战士之怒而闻名。

  他们过着自由奔放的生活，是像骑士这样本分职责原型的自然对立者。他们受个人自由、生存和传统保护的愿望驱使。
