Gurney对世界构建的每个方面都付出了极大的关注，使他的书籍走红全球。 这一切都是为了说，构建世界远非仅仅输入一些提示那么简单！很容易创造出由陈词滥调构成的死气沉沉的地方。但当我们开始思考我们的城市和建筑的文化和历史，我们的居民以及他们的生活时，我们可以开始为我们的城市注入更多生命。
