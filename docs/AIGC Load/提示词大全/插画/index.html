   下面提示词中提到相关专属名词👇：

  • “Slack和Dropbox风格” 都以简洁、现代和用户友好为特点，旨在提供高效的团队协作体验。

  • “Behance风格” 用于展示和分享各种艺术和设计作品，这两个平台并没有固定的单一风格，而是展示了广泛的多样性。这些平台上的作品包括插画、设计、建模、摄影、动画等多个领域。

🌲 19世纪植物插图

   在Pierre-Joseph Redouté和其他植物插图大师的风格中，创作详细而美丽的19世纪植物插图。

   下面提示词中提到相关专属名词👇：

  • “Pierre-Joseph Redoute风格” Pierre-Joseph Redouté（1759年 - 1840年）是一位著名的比利时植物插图画家，被认为是植物插图领域的大师之一。他的风格被描述为精细、精致和逼真。
