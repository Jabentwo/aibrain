前言

   艺术的历史就是人类自身的历史。从描绘当天狩猎场景的洞穴壁画到能够描绘仅受到我们想象力限制的复杂主题的人工智能生成艺术，人类历史的每个时代都被我们对艺术的需求所标记。

   正如你无疑所注意到的，人工智能艺术在2022年迎来了巨大的飞跃。我相信未来的历史书将把这十年的创新视为人类进步中一些最重要的，可以与古腾堡印刷术的重要性相媲美的时刻。

   任何新事物都伴随着学习曲线，使用生成式人工智能工具也不例外。但在这种情况下，随着工具的进步，学习曲线似乎变得越来越平缓。Midjourney最新版本V4的用户都表示，它变得更容易满足你的需求。这种趋势将在未来持续。

   因此，这不是一本告诉你确切使用什么提示的书。随着工具的进步，提示会发生变化。这是一本鼓励你以可能没有考虑到的方式使用这些工具的书。所有的提示都被包含在内，以帮助你更快地进行实验。

   我把这本书分成了主题部分，这样你就可以浏览你最感兴趣的主题。它设计成让你能够迅速翻阅，直到某些内容引起你的注意。

   对于艺术家和艺术爱好者来说，现在是一个令人兴奋的时刻。我们站在一个巨大的悬崖边，没有人知道接下来会发生什么。这是令人振奋、刺激，也有点吓人。

   实际上，我们只是在这里享受这段旅程。我希望这本书能帮助你将你的想象力变成现实。

— Aitu Ma
